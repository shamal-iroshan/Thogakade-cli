# Instructions

## User

### Register
```
--type user register <username> <password> <role>
```

### Login
```
--type user login <username> <password>
```

### View Current User
```
--type user current
```

## Item

### Add
```
--type item add <name> <buyingPrice> <sellingPrice>
```
### View All
```
--type item view-all
```
### View Single Item
```
--type item view <id>
```
